Configuración Básica de Git

Lo primero que tenemos que hacer es crear una carpeta en nuestra computadora e indicarle a git que comience a rastrear los cambios de esa carpeta, los comandos que vamos a utilizar son los siguientes:
mkdir [nombredetucarpeta]
git init
git config --global user.name "tu nombre de usuario de github"
git config --global user.email "tucorreoengithub@correo.com"
