HTML5 funciona con etiquetas, en las cuales utilisamos el símbolo de menor que, el símbolo de mayor que y una diagonal:
<nombre de la etiqueta > etiqueta de apertura.
<& / nombre de la etiqueta > etiqueta de cierre.

La mayoria de las etiquetas vienen en pares, sin embargo algunas vienen solas como por ejemplo:
La primer etiqueta que se coloca al inicio de todo documento de html:
<!DOCTYPE html >
Es la declaración del documento y sirve para indicarle al navegador que se esta utilizando la versión de HTML5.

Estructura básica de una página web:

<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8 />
<title>Hola mundo</>
</head>
<body>
<h1>Bienvenidos</h1>
</body>
</html>
